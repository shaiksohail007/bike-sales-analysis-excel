# bike sales analysis (Europe)

## Project Overview
This project analyzes bike sales data across european countries using Microsoft excel.
this analysis focuses on sales performences by country and year using Pivot Tables and Charts.

## Dataset information 
- Source : kaggle
- data type: secondary (public dataset)
- format: Excel 

## tools used 
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Basic Formatting
## Analysis Performed
-  cleaned and prepared raw data
-  created pivot table for country-wise and year-wise sales
-  Analysed age-wise contributions to bike sales

## Key Insights
-  sales trend comparision across countries
-  adults (35-64) contributed the highest shares of sales
-  simple visual insights using Excel dashboards

## Learning Outcome
-  hands-on experience with Excel Pivot Tables
-  Improvised data analysis and visualization skills
